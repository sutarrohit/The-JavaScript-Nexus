# 1. What are Pseudo-Classes?

### **Definition**

A **pseudo-class** in CSS is used to define a **special state of an element** â€” something that canâ€™t be targeted directly using HTML attributes or regular selectors.

Itâ€™s written with a **colon (`:`)** followed by the pseudo-class name.

**Syntax:**

```css

}
```

---

### **Common Examples**

| Pseudo-Class     | Meaning / When it applies               | Example                                 |
| ---------------- | --------------------------------------- | --------------------------------------- |
| `:hover`         | When user hovers mouse over an element  | `button:hover { background: blue; }`    |
| `:focus`         | When an element (like input) is focused | `input:focus { border-color: green; }`  |
| `:active`        | When an element is being clicked        | `a:active { color: red; }`              |
| `:visited`       | For links that have been visited        | `a:visited { color: purple; }`          |
| `:first-child`   | Selects the first child of a parent     | `li:first-child { font-weight: bold; }` |
| `:last-child`    | Selects the last child of a parent      | `li:last-child { color: gray; }`        |
| `:nth-child(n)`  | Selects nth child                       | `li:nth-child(2) { color: red; }`       |
| `:not(selector)` | Excludes elements matching selector     | `p:not(.highlight) { color: black; }`   |
