# **Transitions**

### **1. transition-property**

**Controls which CSS properties should transition.**

| Class                   | Description                                                                        | Styles                                                            |
| ----------------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| `transition`            | Enables transition on common properties like `color`, `opacity`, `transform`, etc. | Default duration: `150ms`, easing: `cubic-bezier(0.4, 0, 0.2, 1)` |
| `transition-all`        | Transitions all properties                                                         | `transition-property: all;`                                       |
| `transition-colors`     | Transitions only color-related properties                                          | `color, background-color, border-color, fill, stroke...`          |
| `transition-opacity`    | Transitions only opacity                                                           | `transition-property: opacity;`                                   |
| `transition-shadow`     | Transitions only box-shadow                                                        | `transition-property: box-shadow;`                                |
| `transition-transform`  | Transitions transforms (translate, scale, rotate)                                  | `transition-property: transform;`                                 |
| `transition-none`       | Disables transitions                                                               | `transition-property: none;`                                      |
| `transition-[value]`    | Custom value                                                                       | `transition-property: <value>;`                                   |
| `transition-(--custom)` | Uses CSS variable                                                                  | `transition-property: var(--custom);`                             |

**Example:**

```html
<button class="bg-blue-500 transition duration-300 ease-in-out hover:scale-110 hover:bg-indigo-500">
    Save Changes
</button>
```

**Accessibility:**

-   `motion-safe:` â†’ Apply transition only if user allows motion
-   `motion-reduce:` â†’ Disable for reduced motion users

**Responsive Example:**

```html
<button class="transition-none md:transition-all">Click</button>
```

### **2. transition-behavior**

**Controls how transitions behave for discrete (non-continuous) properties.**

| Class                 | Styles                                 |
| --------------------- | -------------------------------------- |
| `transition-normal`   | `transition-behavior: normal;`         |
| `transition-discrete` | `transition-behavior: allow-discrete;` |

**Example:**

```html
<button class="hidden transition-all transition-discrete not-peer-has-checked:opacity-0 peer-has-checked:block">
    I fade out
</button>
```

**Responsive Example:**

```html
<button class="transition-discrete md:transition-normal">Click</button>
```

### **3. transition-duration**

**Controls how long transitions take.**

| Class                      | Styles                          |
| -------------------------- | ------------------------------- |
| `duration-150`             | `transition-duration: 150ms;`   |
| `duration-300`             | `transition-duration: 300ms;`   |
| `duration-700`             | `transition-duration: 700ms;`   |
| `duration-initial`         | `transition-duration: initial;` |
| `duration-[1s]`            | Custom value                    |
| `duration-(--my-duration)` | CSS variable                    |

**Example:**

```html
<button class="transition duration-700 ease-in-out">Button</button>
```

**Responsive Example:**

```html
<button class="duration-0 md:duration-150">Click</button>
```

---

### **4. transition-timing-function**

**Controls easing (speed curve) of transitions.**

| Class              | Easing Function                |
| ------------------ | ------------------------------ |
| `ease-linear`      | `linear`                       |
| `ease-in`          | `cubic-bezier(0.4, 0, 1, 1)`   |
| `ease-out`         | `cubic-bezier(0, 0, 0.2, 1)`   |
| `ease-in-out`      | `cubic-bezier(0.4, 0, 0.2, 1)` |
| `ease-[value]`     | Custom cubic-bezier            |
| `ease-(--my-ease)` | CSS variable                   |

**Example:**

```html
<button class="transition duration-300 ease-in-out">Hover Me</button>
```

**Custom easing:**

```html
<button class="ease-[cubic-bezier(0.95,0.05,0.795,0.035)]">Smooth</button>
```

**Custom Theme Example:**

```css
@theme {
    --ease-in-expo: cubic-bezier(0.95, 0.05, 0.795, 0.035);
}
```

Use in markup:

```html
<button class="ease-in-expo">Button</button>
```

### **5. transition-delay**

**Controls when the transition should start.**

| Class                | Styles                     |
| -------------------- | -------------------------- |
| `delay-150`          | `transition-delay: 150ms;` |
| `delay-300`          | `transition-delay: 300ms;` |
| `delay-700`          | `transition-delay: 700ms;` |
| `delay-[1s]`         | Custom value               |
| `delay-(--my-delay)` | CSS variable               |

**Example:**

```html
<button class="transition delay-300 duration-300 ease-in-out">Button B</button>
```

**Responsive Example:**

```html
<button class="delay-150 md:delay-300">Click</button>
```
