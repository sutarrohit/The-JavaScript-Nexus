## üé® **1. Background Attachment**

**Purpose:**
Controls how a background image behaves when scrolling.

| Class       | Style                            | Description                                                           |
| ----------- | -------------------------------- | --------------------------------------------------------------------- |
| `bg-fixed`  | `background-attachment: fixed;`  | Background image stays fixed relative to the viewport when scrolling. |
| `bg-local`  | `background-attachment: local;`  | Background scrolls with the element‚Äôs content.                        |
| `bg-scroll` | `background-attachment: scroll;` | Background scrolls with the page (default behavior).                  |

**Examples:**

```html
<div class="bg-[url(/img/mountains.jpg)] bg-fixed ..."></div>
<div class="bg-[url(/img/mountains.jpg)] bg-local ..."></div>
<div class="bg-[url(/img/mountains.jpg)] bg-scroll ..."></div>
```

**Responsive Example:**

```html
<div class="bg-local md:bg-fixed ..."></div>
```

**Usage Tip:**
`bg-fixed` gives a **parallax-like** effect; good for hero sections or backgrounds.

## üñºÔ∏è **2. Background Clip**

**Purpose:**
Defines how far the background extends within an element‚Äôs box model.

| Class             | Style                           | Description                                                |
| ----------------- | ------------------------------- | ---------------------------------------------------------- |
| `bg-clip-border`  | `background-clip: border-box;`  | Background extends up to the border (default).             |
| `bg-clip-padding` | `background-clip: padding-box;` | Background is visible only within the padding area.        |
| `bg-clip-content` | `background-clip: content-box;` | Background is limited to the content area.                 |
| `bg-clip-text`    | `background-clip: text;`        | Background is clipped to text (for gradient text effects). |

**Examples:**

```html
<div class="border-4 bg-indigo-500 bg-clip-border p-3"></div>
<div class="border-4 bg-indigo-500 bg-clip-padding p-3"></div>
<div class="border-4 bg-indigo-500 bg-clip-content p-3"></div>
```

**Gradient text:**

```html
<p class="bg-linear-to-r from-pink-500 to-violet-500 bg-clip-text text-transparent font-extrabold text-5xl">
    Hello world
</p>
```

**Responsive Example:**

```html
<div class="bg-clip-border md:bg-clip-padding ..."></div>
```

## üß± **3. Background Color**

**Purpose:**
Sets or customizes the background color of an element.

| Class Example     | Description                     |
| ----------------- | ------------------------------- |
| `bg-blue-500`     | Sets a solid Tailwind color.    |
| `bg-transparent`  | Makes background transparent.   |
| `bg-sky-500/50`   | Adds opacity (50% transparent). |
| `bg-[#50d71e]`    | Custom hex color.               |
| `bg-(--my-color)` | Uses a CSS variable color.      |

**Examples:**

```html
<button class="bg-blue-500 ...">Button A</button>
<button class="bg-sky-500/50 ...">Button B</button>
<div class="bg-[#50d71e] ..."></div>
```

**Hover state:**

```html
<button class="bg-indigo-500 hover:bg-fuchsia-500 ...">Save changes</button>
```

**Responsive Example:**

```html
<div class="bg-blue-500 md:bg-green-500 ..."></div>
```

**Usage Tip:**
Use opacity modifiers (like `/75`) for layered backgrounds and hover variants for interaction feedback.

## üåÖ **4. Background Image**

**Purpose:**
Controls the background image and gradients.

### üîπ Basic Image

```html
<div class="bg-[url(/img/mountains.jpg)] ..."></div>
```

---

### üîπ Linear Gradients

Use direction + color stops.

| Direction Class   | Description                      |
| ----------------- | -------------------------------- |
| `bg-linear-to-r`  | Gradient left ‚Üí right            |
| `bg-linear-to-t`  | Gradient bottom ‚Üí top            |
| `bg-linear-to-bl` | Gradient top-right ‚Üí bottom-left |
| `bg-linear-65`    | Custom angle (65¬∞)               |

**Example:**

```html
<div class="h-14 bg-linear-to-r from-cyan-500 to-blue-500"></div>
```

---

### üîπ Radial Gradients

Create circular or elliptical gradients.

**Examples:**

```html
<div class="size-18 rounded-full bg-radial from-pink-400 from-40% to-fuchsia-700"></div>
<div class="size-18 rounded-full bg-radial-[at_25%_25%] from-white to-zinc-900 to-75%"></div>
```

---

### üîπ Conic Gradients

Gradients that spin around a center point.

**Examples:**

```html
<div class="size-24 rounded-full bg-conic from-blue-600 to-sky-400 to-50%"></div>
<div class="size-24 rounded-full bg-conic/decreasing from-violet-700 via-lime-300 to-violet-700"></div>
```

---

### üîπ Gradient Color Stops

Set multiple color points.

```html
<div class="bg-gradient-to-r from-indigo-500 via-purple-500 to-pink-500 ..."></div>
```

**Custom stop positions:**

```html
<div class="bg-gradient-to-r from-indigo-500 from-10% via-sky-500 via-30% to-emerald-500 to-90%"></div>
```

---

### üîπ Interpolation Modes

Control how gradient colors blend.

| Modifier                                            | Meaning                      |
| --------------------------------------------------- | ---------------------------- |
| `/srgb`                                             | Standard RGB                 |
| `/hsl`                                              | Hue-Saturation-Lightness     |
| `/oklab`, `/oklch`                                  | Perceptual color spaces      |
| `/longer`, `/shorter`, `/increasing`, `/decreasing` | Adjust hue blending behavior |

**Example:**

```html
<div class="bg-linear-to-r/oklab from-indigo-500 to-teal-400"></div>
```

---

### üîπ Remove Background Image

```html
<div class="bg-none"></div>
```

---

### üîπ Custom Gradient Values

```html
<div class="bg-linear-[25deg,red_5%,yellow_60%,lime_90%,teal] ..."></div>
```

Using a CSS variable:

```html

```

---

### üîπ Responsive Example

```html

```

---

### üîπ Theme Customization


Then use:


## üß© **5. Background Origin**

**Purpose:**
Controls how the background image is positioned relative to an element‚Äôs **border**, **padding**, or **content box**.

| Class               | Style                             | Description                                                    |
| ------------------- | --------------------------------- | -------------------------------------------------------------- |
| `bg-origin-border`  | `background-origin: border-box;`  | Background starts from the **outer border** edge.              |
| `bg-origin-padding` | `background-origin: padding-box;` | Background starts from the **inside of the border** (default). |
| `bg-origin-content` | `background-origin: content-box;` | Background starts from the **content area** only.              |

**Examples:**

```html
<div class="border-4 bg-[url(/img/mountains.jpg)] bg-origin-border p-3"></div>
<div class="border-4 bg-[url(/img/mountains.jpg)] bg-origin-padding p-3"></div>
<div class="border-4 bg-[url(/img/mountains.jpg)] bg-origin-content p-3"></div>
```

**Responsive Example:**

```html
<div class="bg-origin-border md:bg-origin-padding ..."></div>
```

**Usage Tip:**
Useful when you want precise control over how background images align with borders and padding (e.g., patterned borders or framed backgrounds).

## üéØ **6. Background Position**

**Purpose:**
Defines **where** the background image is positioned inside an element.

| Class             | Style                                | Description                           |
| ----------------- | ------------------------------------ | ------------------------------------- |
| `bg-top-left`     | `background-position: top left;`     | Image positioned at top-left corner.  |
| `bg-top`          | `background-position: top;`          | Image positioned at top center.       |
| `bg-top-right`    | `background-position: top right;`    | Image positioned at top-right corner. |
| `bg-left`         | `background-position: left;`         | Image positioned at left center.      |
| `bg-center`       | `background-position: center;`       | Image centered (default).             |
| `bg-right`        | `background-position: right;`        | Image positioned at right center.     |
| `bg-bottom-left`  | `background-position: bottom left;`  | Image positioned at bottom-left.      |
| `bg-bottom`       | `background-position: bottom;`       | Image positioned at bottom center.    |
| `bg-bottom-right` | `background-position: bottom right;` | Image positioned at bottom-right.     |

**Examples:**

```html
<div class="bg-[url(/img/mountains.jpg)] bg-top-left"></div>
<div class="bg-[url(/img/mountains.jpg)] bg-center"></div>
<div class="bg-[url(/img/mountains.jpg)] bg-bottom-right"></div>
```

**Custom Position:**

```html
<div class="bg-position-[center_top_1rem] ..."></div>
```

**Using CSS Variable:**

```html
<div class="bg-position-(--my-bg-position) ..."></div>
```

**Responsive Example:**

```html
<div class="bg-center md:bg-top ..."></div>
```

**Usage Tip:**
Combine with `bg-cover` or `bg-contain` for perfect background image alignment in hero sections or cards.

## ‚ôªÔ∏è **7. Background Repeat**

**Purpose:**
Controls whether and how a background image repeats (tiles) across an element.

| Class             | Style                           | Description                                       |
| ----------------- | ------------------------------- | ------------------------------------------------- |
| `bg-repeat`       | `background-repeat: repeat;`    | Repeats both horizontally & vertically (default). |
| `bg-repeat-x`     | `background-repeat: repeat-x;`  | Repeats **horizontally** only.                    |
| `bg-repeat-y`     | `background-repeat: repeat-y;`  | Repeats **vertically** only.                      |
| `bg-repeat-space` | `background-repeat: space;`     | Evenly spaces images, with gaps at edges.         |
| `bg-repeat-round` | `background-repeat: round;`     | Scales images to fit whole numbers of tiles.      |
| `bg-no-repeat`    | `background-repeat: no-repeat;` | Prevents repetition; shows only one image.        |

**Examples:**

```html
<div class="bg-[url(/img/pattern.png)] bg-repeat-x"></div>
<div class="bg-[url(/img/pattern.png)] bg-no-repeat bg-center"></div>
<div class="bg-[url(/img/pattern.png)] bg-repeat-round"></div>
```

**Usage Tip:**
Use `bg-no-repeat` with `bg-cover` for full-screen images, and `bg-repeat` or `bg-repeat-x/y` for seamless patterns.

## üñºÔ∏è **8. Background Size**

**Purpose:**
Controls how background images scale and fit within their container.

| Class                         | Style                                      | Description                                       |
| ----------------------------- | ------------------------------------------ | ------------------------------------------------- |
| `bg-auto`                     | `background-size: auto;`                   | Keeps image at its natural size.                  |
| `bg-cover`                    | `background-size: cover;`                  | Image fills the container, cropping if necessary. |
| `bg-contain`                  | `background-size: contain;`                | Fits image within container without cropping.     |
| `bg-size-[<value>]`           | `background-size: <value>;`                | Custom background size (e.g., `auto_100px`).      |
| `bg-size-(<custom-property>)` | `background-size: var(<custom-property>);` | Uses a CSS variable.                              |

**Examples:**

‚úÖ **Cover entire container (cropping if needed):**

```html
<div class="bg-[url(/img/mountains.jpg)] bg-cover bg-center"></div>
```

‚úÖ **Fit inside without cropping:**

```html
<div class="bg-[url(/img/mountains.jpg)] bg-contain bg-center"></div>
```

‚úÖ **Keep default size:**

```html
<div class="bg-[url(/img/mountains.jpg)] bg-auto bg-center bg-no-repeat"></div>
```

‚úÖ **Custom value:**

```html
<div class="bg-size-[auto_100px] ..."></div>
```

‚úÖ **CSS variable:**

```html
<div class="bg-size-(--my-image-size) ..."></div>
```

‚úÖ **Responsive Example:**

```html
<div class="bg-auto md:bg-contain lg:bg-cover ..."></div>
```

**Usage Tip:**
Use `bg-cover` for full-width banners, `bg-contain` for icons or logos, and `bg-auto` when you need natural image sizing.
