## âš–ï¸ 4. **Key Differences Between SQL and NoSQL**

| Feature               | SQL (Relational)                   | NoSQL (Non-relational)                      |
| --------------------- | ---------------------------------- | ------------------------------------------- |
| **Data Model**        | Tables (rows and columns)          | Documents, key-value pairs, graphs, columns |
| **Schema**            | Fixed, predefined                  | Flexible, dynamic                           |
| **Scalability**       | Vertical (add power to one server) | Horizontal (add more servers/nodes)         |

| **Best For**          | Structured data, transactions      | Unstructured/semi-structured data           |
| **Examples**          | MySQL, PostgreSQL, Oracle          | MongoDB, Cassandra, Redis, Neo4j            |
