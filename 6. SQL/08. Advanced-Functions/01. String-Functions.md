## üß† **Advanced SQL Functions**

### üîç **Introduction**

**Advanced SQL functions** go beyond basic querying to perform **data transformation, analysis, and formatting** directly within SQL statements.
They allow you to:

-   Manipulate text and numerical data
-   Perform date and time operations
-   Aggregate and summarize information
-   Write complex analytical queries (e.g., ranking, windowing, etc.)

---

### ‚öôÔ∏è **Categories of Advanced SQL Functions**

1. **String Functions** ‚Üí Work on text data (e.g., `CONCAT`, `LENGTH`, `UPPER`, `LOWER`, `REPLACE`, `SUBSTRING`)
2. **Numeric Functions** ‚Üí Perform mathematical operations (e.g., `ROUND`, `ABS`, `CEIL`, `FLOOR`)
3. **Date and Time Functions** ‚Üí Handle date/time values (e.g., `NOW()`, `DATE_ADD`, `EXTRACT`)
4. **Aggregate Functions** ‚Üí Summarize data (e.g., `SUM`, `AVG`, `COUNT`, `MIN`, `MAX`)
5. **Window (Analytical) Functions** ‚Üí Compute results across sets of rows (e.g., `ROW_NUMBER()`, `RANK()`, `OVER()`)
6. **Conditional Functions** ‚Üí Execute logic in queries (e.g., `CASE`, `COALESCE`, `NULLIF`)

---

## üß© **String Functions in SQL**

String functions are used to **manipulate, search, or modify text (character) data** stored in columns.
They are extremely useful for formatting output or preparing data for reports.

Let‚Äôs explore the most important ones üëá

### 1. üîó **`CONCAT()` ‚Äì Concatenate Strings**

**Purpose:**
Combines two or more strings into one.

**Syntax:**

```sql
CONCAT(string1, string2, string3, ...)
```

**Example:**

```sql
SELECT CONCAT('Hello', ' ', 'World') AS greeting;
```

‚úÖ **Output:**

```
greeting
---------
Hello World
```

**Example with table data:**

```sql
SELECT CONCAT(first_name, ' ', last_name) AS full_name
FROM students;
```

---

### 2. üìè **`LENGTH()` ‚Äì Find String Length**

**Purpose:**
Returns the number of characters in a string.

**Syntax:**

```sql
LENGTH(string)
```

**Example:**

```sql
SELECT LENGTH('Database') AS length;
```

‚úÖ **Output:**

```
length
-------
8
```

**Use case:**
Check which usernames are too short:

```sql
SELECT username
FROM users
WHERE LENGTH(username) < 5;
```

---

### 3. ‚úÇÔ∏è **`SUBSTRING()` ‚Äì Extract Part of a String**

**Purpose:**
Extracts a portion (substring) of a given string based on position and length.

**Syntax:**

```sql
SUBSTRING(string, start_position, length)
```

**Example:**

```sql
SELECT SUBSTRING('PostgreSQL', 1, 4) AS result;
```

‚úÖ **Output:**

```
result
-------
Post
```

**Another example:**

```sql
SELECT SUBSTRING('HelloWorld', 6) AS result;
```

‚úÖ **Output:**

```
result
-------
World
```

---

### 4. ü™Ñ **`REPLACE()` ‚Äì Replace Substring**

**Purpose:**
Replaces occurrences of a substring within a string with another substring.

**Syntax:**

```sql
REPLACE(original_string, substring_to_replace, replacement_string)
```

**Example:**

```sql
SELECT REPLACE('I like SQL', 'SQL', 'PostgreSQL') AS new_string;
```

‚úÖ **Output:**

```
new_string
------------
I like PostgreSQL
```

**Use case in tables:**

```sql
SELECT REPLACE(email, 'gmail.com', 'outlook.com') AS updated_email
FROM users;
```

---

### 5. üî† **`UPPER()` ‚Äì Convert to Uppercase**

**Purpose:**
Converts all characters in a string to uppercase.

**Syntax:**

```sql
UPPER(string)
```

**Example:**

```sql
SELECT UPPER('sql functions') AS upper_case;
```

‚úÖ **Output:**

```
upper_case
------------
SQL FUNCTIONS
```

**Use case:**

```sql
SELECT UPPER(first_name) AS capital_name FROM employees;
```

---

### 6. üî° **`LOWER()` ‚Äì Convert to Lowercase**

**Purpose:**
Converts all characters in a string to lowercase.

**Syntax:**

```sql
LOWER(string)
```

**Example:**

```sql
SELECT LOWER('HELLO WORLD') AS lower_case;
```

‚úÖ **Output:**

```
lower_case
------------
hello world
```

**Use case:**
Ensure case-insensitive comparisons:

```sql
SELECT *
FROM users
WHERE LOWER(email) = LOWER('ADMIN@EXAMPLE.COM');
```

## üí° **Practical Tip**

String functions are often **combined** for powerful transformations.
Example:

```sql
SELECT
    UPPER(REPLACE(SUBSTRING(name, 1, 5), 'a', '@')) AS modified_name
FROM users;
```

This:

1. Takes first 5 letters of `name`
2. Replaces all ‚Äòa‚Äô with ‚Äò@‚Äô
3. Converts result to uppercase
